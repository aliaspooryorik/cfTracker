CFTracker (David Boyer)
======================
This is a simple CFC that provides safe and easy access to application / session / memory information.
Using this CFC you can retrieve information from any other app or session.  Along with current coldfusion
memory usage.

======================
ColdFusion Admin Extension
----------------------
Within the cfide subfolder is an extension that provides access to cftracker features from within the CF Admin
